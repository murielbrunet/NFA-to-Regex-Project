Practice Problems for NFA to Regex

Here are some practice problems.
Try to convert the NFA to a regular expression.
Read the captions for guidance.

*Disclaimer: There are many other ways to write regular expressions so don't dispair if your final regex is not the as ours!

Practice Problem 1


Given this NFA, we will create a regular expression.


First, add the Kleene star to arcs looping back to the same state.


Next, delete state 0 by combining the arc leading to state 1 with the arc leading to state 2. The plus sign means one or more. Don't forget to add the loop from state 0 to state 1!


Now, collapse state 1 into state 2 and add their arches together. In this case, simply add the b at the end of each arc expression. Don't forget the loop!


Combine state 2 and 3 and simplify your final expression. We use an or because there were two possible arcs to reach state 3. We can combine the b's the final b* to create at least one or more b's.

Practice Problem 2


This is quite a tricky problem, see if you can figure it out!


Delete state 1, being careful to note the loops it is part of and adding them to state 0 and state 2. Don't forget to add their influence to the two remaining arcs connecting states 0 and 2.


Next, delete state 2 and combine the remaining regex. Once again consider state 2 as a transition in part of a bigger loop. Combine the regex that surround it.


Finally, add the loop that was present on state 0.

Practice Problem 3


Another challenge for you.


First, as always, add the Kleene star to the loop on state 0. You can also write aUb as an alternative to a*b*.


Now combine the loop on state 0 with the arc from state 0 to state 1.


Finally, combine the arc from state 0 to state 1 and the loop on state 1. Presto!
